# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using Python, MySQL, and Power BI to help a retail company understand purchasing patterns, customer engagement, and revenue trends.

The project follows an end-to-end Data Analytics workflow:

- Data Cleaning (Python)
- Database Management (MySQL)
- Business Analysis (SQL)
- Dashboard Development (Power BI)

---

## 🎯 Business Problem

A retail company wants to understand customer shopping behavior to improve:

- Customer engagement
- Product strategy
- Marketing campaigns
- Revenue generation
- Customer loyalty

The objective is to discover actionable insights from customer purchase data.

## 🔄 Project Workflow

### Step 1 — Data Collection

Customer shopping dataset containing:

- Customer demographics
- Product information
- Purchase amount
- Reviews
- Discounts
- Shipping methods
- Payment methods
- Subscription status
- Purchase history

---

### Step 2 — Data Preparation

Performed using Python.

Tasks completed:

- Loaded CSV dataset
- Checked missing values
- Verified data types
- Cleaned dataset
- Exported cleaned data into MySQL using SQLAlchemy

---

### Step 3 — SQL Business Analysis

Performed multiple business queries including:

- Revenue by gender
- Customers spending above average
- Highest rated products
- Shipping comparison
- Subscriber spending analysis
- Discount analysis
- Customer segmentation
- Top purchased products
- Repeat buyer analysis
- Revenue by age group

---

### Step 4 — Power BI Report

Created an interactive dashboard containing:

✔ Number of Customers

✔ Average Purchase Amount

✔ Average Review Rating

✔ Revenue by Category

✔ Sales by Category

✔ Revenue by Age Group

✔ Sales by Age Group

✔ Subscription Analysis

✔ Interactive Filters

---

## 📈 Key Insights

- Clothing category generated the highest revenue.
- Young Adults contributed the highest sales.
- Subscription customers showed different purchasing behavior.
- Review ratings helped identify top-performing products.
- Discount usage varied significantly across products.
- Customer segmentation identified Loyal, Returning, and New customers.

## 🚀 How to Run

### Clone Repository

```bash

```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Open Notebook

```

```

### Import SQL

Run


inside MySQL Workbench.


using Power BI Desktop.
